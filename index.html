Parse Error (Syntax Error)

When there is error in the syntax, by then the code execution will stop. Most common reasons are 1- Missing semicolon 2- unclosed braces or quotes

Fatal Error

Fatal errors are when the code is correct but it can't be done, probably because of undefined variable or function. fatal errors stop the code execution

Warning Error

Warning errors will not stop the execution, and they are most likely because of deprecated usage of code or bad practice code, missing file, wrong number of parameters in a function

Notice Error

these errors don't stop the execution of the code as well, they most likely because of trying to access an undefined variable.
